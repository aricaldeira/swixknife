Unidades Xastadári Derivadas

Ainda que seja possível descrever todas as quantidades físicas, como área, volume, velocidade etc. usando apenas combinações das nove unidades fundamentais, usar nomes específicos para cada combinação e caso de uso tem as vantagens de tanto ser uma forma mais curta e direta, quanto, no caso de fenômenos diferentes que são quantificados usando a mesma combinação de unidades fundamentais, comunicar imediatamente tanto a quantidade quanto o quê exatamente está sendo medido;

Um nome único e simples para o que algumas vezes seria uma longa combinação de unidades fundamentais torna também o uso de prefixos mais objetivo e direto, especialmente com medidas como área e volume, por exemplo;

Na lista abaixo se repetem as unidades fundamentais, exceto pela unidade de tempo, que tem sua página de unidades derivadas dedicada, apenas por consistência e facilidade de referência, e são adicionadas algumas unidades derivadas:
