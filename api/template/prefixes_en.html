Shastadari Prefixes

Shastadari prefixes are a set of sezimal (base six) prefixes, originally designed to be used with the Shastadari Units;

For the online conversion between Shastadari units and prefixes, and the units and prefixes of the S.I. and other systems, access the Sezimal Calculator;

Shastadari prefixes are formed by combining the letter for each digit, and in some cases the value, of the exponent correspondent to the prefix, in upper case when the prefix is positive, and lower case when negative.

Prefixes’ names are formed by combining an affix, corresponding to the exponent number/value, plus an infix indicating whether the exponent is positive or negative.
