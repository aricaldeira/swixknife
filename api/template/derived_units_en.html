Shastadari’s Derived Units

Even though it is possible to describe all physical quantities, like area, volume, speed etc. using only combinations of the nine base units, using specific names for each combination and use has the advantages of both being shorter and more direct, and, in the case of different phenomena being quantified using the same combination of base units, a separate name conveys both the quantity and what exactly is being measured;

A singular name for a sometimes long base units’ combination also makes using prefixes straightforward and unambiguous, specially with measures like area and volume;

In the list below we repeat the base units, except for time, which has its dedicated page, just for consistency and easy referencing, and add some derived units:
