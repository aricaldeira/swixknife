Year Format Tags

#[1][2][3][x]
# is mandatory, it starts the formatting tag;
[x] is mandatory, it has to be on of the following:
    Y year with Sezimal Holocene Epoch (+ 200,000): {{ date.format('#y -> #Y') }}
    sY plain year: {{ date.format('#sy -> #sY') }}
    shY year with Holocene Epoch (+ 10,000 ₁₀): {{ date.format('#shy -> #shY') }}

1 is optional, it may be one of the following:
    9 forces use of base ten (default is base six)
    ↋ forces use of base twelve (default is base six)
    Z compresses base six into base thirty-six, using letters as digits
    @ compresses base six into base thirty-six, using regularized digits (with diacritics)
    ! uses Shastadari digits (only for base six)
    @! same as @, but using Shastadari digits

2 is optional, it’s used to force a particular digits group separator; it may be one of the following:
    narrow no-break space (\u202f) - ISO’s default
    a full stop . (\u002e)
    a comma , (\u02c)
    an underscore _ (\u005f)
    a vertical apostrophe ' (\u0027)
    a right single quotation mark/typographical apostrophe ’ (\u2019)
    a modifier letter apostrophe ʼ (\u02bc)
    a dot abot ˙ (\u02d9)
    a plain space (\u0020)
    a non-break space (\u00a0)
    en quad space (\u2000)
    em quad space (\u2001)
    en space (\u2002)
    em space (\u2003)
    three-per-em space (\u2004)
    four-per-em space (\u2005)
    six-per-em space (\u2006)
    figure space (\u2007)
    punctuation space (\u2008)
    thin space (\u2009)
    hair space (\u200a)
    medium mathematical space (\u205f)
    Shastadari’s shadara (6⁶) separator 󱹬 (\U000f1e6c)
    Shastadari’s arda (6³) separator 󱹭 (\U000f1e6d)

3 is optional, it may only be:
    > uses only the last digts: 213212 -> 212, only for years between 213100 and 213555, otherwise it uses the full year number
    > uses only the last digts: 2024 -> 24, only for years between 2010 and 2099, otherwise it uses the full year number
{% endif %}

Text Format Tags

#[1][2][x]
# is mandatory, it starts the formatting tag;
[x] is mandatory, it has to be on of the following:
    M month name
    W weekday name
    O locale’s ordinal number suffix;
        some locales use the ordinal suffix only for the 1st day
        of the month, others, with any date;
[1] is optional, it may be one of the following:
    @ use locale’s abbreviated form for the weekday or month name;
      e.g. French abbreviations:
          lundi, mardi, mercr., jeudi, vendr., sam., dim.
          janv., févr., mars, avr., mai, juin, juill. etc.
    3 use the first 3 characters;
      e.g. French:
          lun, mar, mer, jeu, ven, sam, dim
          jan, fév, mar, avr, mai, jui, jui etc.
    2 use the first 2 characters;
      e.g. French:
          lu, ma, me, je, ve, sa, di
          ja, fé, ma, av, ma, ju, ju etc.
    1 use only the first character;
      e.g. French:
          l, m, m, j, v, s, d
          j, f, m, a, m, j, j etc.
[2] is optional, it may be one of the following:
    ! forces all uppercase result
    ? forces all lowercase result
    > forces title case result
    otherwise, the locale’s case is used as is, for example:
        French defaults to all lowercase: lundi, mardi etc.
        German defaults to all title case: Montag, Dienstag etc.
