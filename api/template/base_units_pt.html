As 13 Unidades Fundamentais
do Sistema Xastadári

As nove unidades fundamentais do sistema de unidades Xastadári são descritas abaixo;

Para cada unidade, o título contém o que é medido se usando aquela unidade, e o nome da unidade seguido pelo seu símbolo;

Também se encontra a palavra que deu origem ao nome da unidade, em escrita devanágari e sua romanização padrão, bem como a pronúncia original descrita se usando o AFI, e o significado original da palavra;

Na sequência, o fenômeno ou propriedade física que deu origem à própria definição da unidade, e, para algumas unidades, uma definição equivalente usando os mesmos parâmetros ou bases da unidade correspondente no S.I., para comparação;

Por fim, o valor de conversão da unidade para e a partir do S.I., tanto em base sezimal quanto em decimal, e, quando relevante, também a conversão entre uma unidade correspondente no sistema de unidades imperial/usuais dos EUA;

São mencionadas algumas unidades derivadas, e, para cada primeira menção a essas unidades, um link a referenciar a origem do nome da unidade derivada é fornecido, assim como também a sua descrição usando somente as unidades base;

Algumas conversões de unidade são apresentadas usando prefixos; entenda melhor como são usados na página Prefixos;
